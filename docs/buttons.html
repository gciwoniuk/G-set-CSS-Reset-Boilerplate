Buttons

Some easy to use button mixins here. Just do .btn-flat(),.btn-gradient() or .btn-plain() with the optional params of color, border thickness, border radius.

Colors

Different colors are added by passing in a color to the .btn-gradient(),.btn-flat(),.btn-plain() mixins.

Sizes

Different sizes are added by adding .btn-large or .btn-small.
